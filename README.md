## Project Overview

**CookIT** is an advanced culinary web platform engineered to bridge the gap between ingredient management and recipe discovery. By integrating Retrieval-Augmented Generation (RAG) with the Google Gemini API, the application offers a context-aware AI assistant capable of generating precise culinary instructions and personalized meal recommendations directly in the browser.

The system utilizes a high-performance FastAPI backend coupled with a responsive React frontend, ensuring a seamless user experience. Data retrieval is optimized using ChromaDB, allowing for semantic search capabilities that go beyond simple keyword matching.

## Key Features

* **Generative AI Integration**: Leverages Google Gemini API to create bespoke recipes and cooking advice dynamically.
* **RAG Architecture**: Implements Retrieval-Augmented Generation to ground AI responses in valid culinary data, reducing hallucinations and improving accuracy.
* **Semantic Search**: Utilizes ChromaDB to perform vector-based searches, enabling users to find recipes based on vague descriptions or ingredient clusters.
* **Reactive Web Interface**: A modern, component-based UI built with React for fluid navigation and interaction.
* **Optimized Performance**: Backend services architected with FastAPI for asynchronous processing and low-latency responses.
